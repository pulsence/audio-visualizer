# Commit Message Format

This document defines the required commit message format for Audio Visualizer.

## Commit Message Template

```
Added [feature name]

Feature Name:
- Bullet point describing capability
- Another capability
- Configuration options added
```

## Guidelines

- Use present tense ("Added" not "Adding")
- First line is a brief summary of the change
- Follow with a blank line, then bullet points describing:
  - What capability was added
  - Any other notable changes
  - Configuration options that were added/modified

## Example

```
Added chroma force circle visualizer

Chroma Force Circle Visualizer:
- Added physics-based circle visualizer driven by chromagram data
- Added gravity and force strength settings
- Added color mode support (single, gradient, per-band)
```
